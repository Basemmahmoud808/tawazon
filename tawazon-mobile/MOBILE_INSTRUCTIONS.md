# 📱 دليل تشغيل تطبيق توازن على الهواتف (أندرويد وآيفون) 🌿

تم عزل ملفات التطبيق بالكامل داخل مجلد `tawazon-mobile` لكي يظل موقعك الأصلي يعمل بشكل مستقل وآمن دون أي تغيير.

إليك الخطوات المفصلة لتشغيل وبناء التطبيق على هاتفك:

---

## 🟢 1. لتشغيل تطبيق الأندرويد (Android)

### المتطلبات الأساسية:
- تحميل وتثبيت برنامج **[Android Studio](https://developer.android.com/studio)** (مجاني).

### خطوات التشغيل:
1. افتح موجه الأوامر (Terminal) في مجلد `tawazon-mobile` ونفّذ الأمر التالي لفتح المشروع في Android Studio تلقائياً:
   ```bash
   npx cap open android
   ```
   *أو يمكنك فتح برنامج Android Studio يدوياً، واختيار **Open Project** ثم تصفح واختيار المجلد المسمى `android` الموجود داخل `tawazon-mobile`.*

2. انتظر ثوانٍ حتى ينتهي البرنامج من تحميل إعدادات Gradle.
3. قم بتوصيل هاتفك بالكمبيوتر عبر كابل USB وتأكد من تفعيل **وضع تصحيح أخطاء USB (USB Debugging)** في إعدادات المطور بهاتفك.
4. اضغط على زر **Run (السهم الأخضر)** بالأعلى في Android Studio ليتم تثبيت التطبيق وتفعيله فوراً على هاتفك!
5. لإخراج ملف الـ APK النهائي للنشر، اختر من القائمة بالأعلى: **Build -> Build Bundle(s) / APK(s) -> Build APK(s)**.

---

## 🍎 2. لتشغيل تطبيق الآيفون (iOS)

### المتطلبات الأساسية:
- جهاز كمبيوتر يعمل بنظام **Mac (macOS)**.
- تحميل وتثبيت برنامج **Xcode** من متجر الماك (App Store).

### خطوات التشغيل:
1. افتح الـ Terminal في مجلد `tawazon-mobile` ونفّذ الأمر التالي لفتح المشروع في Xcode تلقائياً:
   ```bash
   npx cap open ios
   ```
   *أو افتح مجلد `tawazon-mobile/ios/App` وافتح الملف الذي ينتهي بـ `.xcworkspace`.*

2. في Xcode، اختر الهاتف المحاكي (Simulator) أو قم بتوصيل هاتف iPhone الخاص بك عبر الكابل.
3. اضغط على زر **Run (المثلث أو Play)** بالركن الأيسر العلوي لتثبيت وتشغيل التطبيق على الهاتف مباشرة.

---

## 🔄 3. تحديث الكود ومزامنة التغييرات (Developer Workflow)

إذا قمت بتعديل أي كود في المستقبل داخل مجلد `tawazon-mobile/src` وتريد أن تظهر هذه التعديلات في التطبيق، قم بتنفيذ الأوامر التالية:

1. بناء ملفات الويب المحدثة:
   ```bash
   npm run build
   ```
2. مزامنة التعديلات مع تطبيقات الهاتف:
   ```bash
   npx cap sync
   ```

ثم قم بإعادة تشغيل التطبيق من Android Studio أو Xcode لتظهر التحديثات فوراً! 🎉
